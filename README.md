# Crypto Tracker

Real-time cryptocurrency price tracker using the CoinGecko API with live search and auto-refresh.

## Features
- Live prices for top 24 cryptocurrencies
- 24h price change indicators (green/red)
- Market cap and volume data
- Live search/filter
- Auto-refresh every 60 seconds
- Graceful fallback to mock data when API is unavailable

## Usage
```bash
# Open index.html in a browser — no build step needed
open index.html
```

## Tech Stack
- Vanilla JavaScript (ES6+)
- CoinGecko Public API
- CSS3 with custom properties

## API
Uses the free [CoinGecko API](https://www.coingecko.com/api/documentation) — no API key required.
